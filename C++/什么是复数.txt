
复数是由实部和虚部组成的一个数。在数学中，复数可以表示为 a + bi 的形式，其中 a 和 b 都是实数，i 表示虚数单位，满足 i^2 = -1。

一个复数 a + bi 可以表示为一个有序数对 (a, b)，也可以表示为一个向量 (a, b) 在平面直角坐标系中的位置，其中实部 a 对应于横轴上的位置，虚部 b 对应于纵轴上的位置。这种用向量描述复数的方式被称为极坐标形式。

复数之间的加、减、乘、除运算都可以通过展开实部和虚部的方式来计算得出。例如，两个复数 a + bi 和 c + di 的和为：

(a + bi) + (c + di) = (a + c) + (b + d)i

而它们的积为：

(a + bi)(c + di) = (ac - bd) + (ad + bc)i

复数还有共轭复数的概念，一个复数的共轭复数表示将其虚部取相反数的复数。例如，一个复数 a + bi 的共轭复数为 a - bi。

复数在数学中具有广泛的应用，可以解决一些无法在实数范围内解决的问题，例如方程 x^2 + 1 = 0 就没有实数解，但可以用复数解 a + bi 来表示。同时，复数在物理、工程、计算机等领域也有广泛的应用。
